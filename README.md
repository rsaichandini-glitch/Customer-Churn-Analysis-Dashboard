# Customer Churn Analysis Dashboard | Power BI

## Project Overview

This project analyzes customer churn using the IBM Telco Customer Churn dataset. The dashboard helps identify factors influencing customer churn and provides actionable insights to improve customer retention.

---

## Objectives

- Analyze customer churn patterns.
- Calculate overall churn rate.
- Identify high-risk customer segments.
- Compare churn across contract types, payment methods, and internet services.
- Provide business insights for improving customer retention.

---

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel (CSV Dataset)

---

## Dataset

**Dataset:** IBM Telco Customer Churn

The dataset contains customer information including:
- Customer ID
- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Contract Type
- Payment Method
- Internet Service
- Monthly Charges
- Total Charges
- Churn Status

---

## Dashboard Features

### KPI Cards
- Total Customers
- Churned Customers
- Active Customers
- Churn Rate (%)
- Average Monthly Charges
- Average Customer Tenure

### Visualizations
- Churn by Contract Type
- Churn by Payment Method
- Churn by Internet Service
- Churned Customers by Gender
- Average Monthly Charges by Churn
- Average Tenure by Churn
- Churn by Senior Citizen

### Interactive Filters
- Gender
- Contract Type
- Internet Service
- Payment Method

---

## Key Insights

- Month-to-month contracts have the highest churn rate.
- Customers with higher monthly charges are more likely to churn.
- Customers with longer tenure tend to stay with the company.
- Contract type and payment method significantly influence customer retention.
